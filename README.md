# LinkedIn Thought Leadership Agent

## Overview

The LinkedIn Thought Leadership Agent is an AI-powered workflow built in n8n using Google's Gemini model.

The agent analyzes professional content, identifies key insights, extracts AI and automation opportunities, and generates LinkedIn-ready commentary in a custom voice.

This project demonstrates practical AI automation, prompt engineering, and workflow design using modern no-code tools.

## Features

* AI-powered content analysis
* Professional insight extraction
* AI and automation opportunity detection
* LinkedIn repost generation
* Custom voice and tone configuration
* Built with n8n and Gemini

## Workflow

User Input → AI Agent → Content Analysis → AI Take → LinkedIn Repost

## Technology Stack

* n8n
* Google Gemini
* AI Agent Node

## Example Output

The agent generates:

* Summary
* Why It Matters
* AI/Automation Take
* LinkedIn Repost

## Use Case

This workflow helps professionals quickly transform industry content into thought leadership posts focused on AI, automation, and workflow optimization.
